## 📝 Typography System

### Font Hierarchy & Medical Readability
```
Primary Typeface: Roboto
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

H1 - Main Headings
Size: 24px | Weight: Medium | Color: #2E86AB
Line Height: 32px | Letter Spacing: 0.25px
Usage: Screen titles, main headings

H2 - Section Headings  
Size: 20px | Weight: Medium | Color: #424242
Line Height: 28px | Letter Spacing: 0.15px
Usage: Section dividers, category headers

H3 - Subsection Headings
Size: 18px | Weight: Regular | Color: #424242
Line Height: 24px | Letter Spacing: 0.1px
Usage: Card titles, subsections

Body - Main Content
Size: 16px | Weight: Regular | Color: #424242
Line Height: 24px | Letter Spacing: 0.5px
Usage: General content, descriptions

Caption - Supporting Text
Size: 14px | Weight: Regular | Color: #757575
Line Height: 20px | Letter Spacing: 0.25px
Usage: Labels, timestamps, metadata

Medical Terms - Terminology
Size: 16px | Weight: Regular | Font: Roboto Mono
Color: #424242 | Background: #F5F5F5
Usage: Drug names, medical conditions, measurements
```

### Accessibility Compliance
- **Minimum text size**: 14px for all body text
- **Contrast ratio**: 4.5:1 for normal text, 3:1 for large text
- **Font weight**: Medium (500) minimum for headings
- **Line spacing**: 1.5x minimum for readability
- **Touch targets**: 44px minimum for interactive elements

```
• Duration: 1.5 seconds continuous loop
• Easing: Linear
• Direction: Left to right shimmer

```

### Transition Effects
```
Page Transitions:
• Duration: 300ms
• Easing: Ease-out cubic-bezier(0.4, 0, 0.2, 1)
• Type: Slide (iOS) / Fade + Scale (Android)
• Direction: Left to right for forward navigation

Button Press Feedback:
• Duration: 150ms
• Effect: Scale (0.95) + Shadow reduction
• Recovery: 200ms ease-out
• Color: Subtle darker shade (10% opacity overlay)

Progress Bar Animation:
• Duration: 500ms per segment
• Easing: Ease-out
• Fill: Smooth left-to-right progression
• Color transition: Gradient effect on fill
```

### Success/Error Feedback
```
Success Animation (Checkmark):
     ╭───╮
    ╱  ✓  ╲  ← Scale in from 0 to 1
   │       │     Duration: 400ms
    ╲     ╱      Easing: Bounce
     ╰───╯

Error Animation (Shake):
[Button] ←→←→←  ← Horizontal shake
• Amplitude: 8px
• Frequency: 3 cycles
• Duration: 400ms
• Easing: Ease-out
```
